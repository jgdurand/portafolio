Sistema web Laboratorio Clinico

Sistema web creado en el año 2016 netamente con los lenguajes HTML,CSS,JS,PHP,SQL. El sistema esta desarrollado para el control de pacientes y resultados de laboratorios clinicos, contando con acceso a multiples usuarios y permisos, registro de paciente y resultados, posibilidad de filtrar pacientes por empresas y herramientas necesarias para el control de un laboratorio.

Desarrollado Agosto 2016

Sistema web Almacén y Ventas

Sistema web desarrollado en 2017, diseño basado en Framework de Bootstrap y Jquery. Está desarrollado para administración, almacen y venta de productos. El sistema posee acceso a multiples usuarios y permisos, almacén e inventario de productos, reporte de ventas mensuales y diarias, detalles de los pagos realizados en las ventas, historial de actividades, seccion de caja para venta de productos.

Desarrollado Mayo 2017

Sistema web Balance e Inventario

Sistema web desarrollado en 2018, diseño basado en Framework de Bootstrap y Jquery. Está desarrollado para el control y administración economica. Permitiendo al usuario monitorear entradas y salidas de dinero en multiples sucursales o negocios. Poseé acceso a multiples usuarios y permisos, balance de entrada y salida de dinero e inventario de productos.

Desarrollado Julio 2018

Asclepio Software Medico

Asclepio es un proyecto personal, está pensado en ser un sistema para el control y administración de consultorios medicos. Su caracteristica principal; está pensado en que sea completamente personalizable, ademas de ser adaptable a las especialidades del médico. El sistema contaría con acceso a multiples usuarios y permisos, control y administracion de pacientes e historias medicas, gestion de consultas, gestion para ecografías y estudios, balance economico de entrada y salida, inventario de articulos, agenda médica y estadisticas mensuales y diarías.
